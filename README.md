🎨 Air Canvas

Air Canvas is a real-time gesture-controlled drawing application that enables users to draw on a virtual canvas using hand movements captured through a webcam. The project integrates computer vision and deep learning techniques to create a touchless and interactive drawing system.

📌 Project Overview

Air Canvas captures live video input and processes each frame to detect and track hand landmarks in real time. Using gesture recognition powered by Convolutional Neural Networks (CNNs), the system interprets finger movements and converts them into digital strokes on the screen.
This eliminates the need for physical input devices such as a mouse, stylus, or touchscreen.

🚀 Key Features

Real-time hand tracking and landmark detection
Gesture recognition using Convolutional Neural Networks (CNN)
Deep learning implementation using TensorFlow and Keras
Smooth and responsive virtual drawing
Multiple color selection options
Canvas clear/reset functionality
Touchless human-computer interaction

🛠️ Tech Stack

🧠 Technical Implementation

Webcam frames are captured using OpenCV
Hand landmarks are extracted using MediaPipe
A CNN model is built and trained using TensorFlow/Keras for gesture classification
Real-time predictions are mapped to drawing commands
The system integrates deep learning inference with interactive UI logic
This architecture combines classical computer vision techniques with deep learning-based gesture recognition to build an intelligent, responsive system.
